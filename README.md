## API Reference

#### Get all products (for exercise 3)

```http
  GET https://fakestoreapi.com/products
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `id` | `int` | element id |
| `title` | `string` | product title |
| `price` | `double` | product price |
